SIMPL_URL

A URL Shortener Microservice

Manual

Usage Example

Instead of sending the URL to shorten in above form, you can also send it as a POST request to
https://admijw-simplurl.herokuapp.com/api/shorturl/new/[your_url]

Example: 

fetch('./api/shorturl/new', {
    method: 'POST',
    body: {desired-link-to-shorten}
})
.then(res => res.json() )
.then(res => console.log(res));

Either send in the body as raw-text, or x-www-form-urlencoded data with key name as url

Result

Upon successful shortening, the server will return a json.

Example:
{ original_url : 'https://freeCodeCamp.org', short_url : aBc}
where you can redirect to the target url via
/v/<short_url>
or
/api/shorturl/<short_url>

On the other hand, if error occurs, the json will look like:
{ error: <error-message> }

Notes

  • Due to limited database, your shortened url may expire after being inactive for 3 days, depending on request volume
  • Due to the short URL generation algorithm used, URL shortening may fail when you try to shorten it. You may retry several times
